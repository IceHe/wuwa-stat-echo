# 鸣潮声骸副词条文档

## 作用

前端通过 `src/stores/constants.ts` 内置了鸣潮声骸副词条的展示字典，用于：

- 渲染按钮文本
- 设置颜色
- 计算位图
- 生成调谐记录提交参数

## 前端中的表示方式

### 副词条对象

前端会为每类副词条维护：

- `num`
- `name`
- `font_color`
- `bitmap`

### 档位对象

前端会为每个档位维护：

- `substat_number`
- `value_number`
- `desc`
- `desc_full`

## 位图规则

前端和后端统一使用位图：

- 类型位图：`1 << substat_number`
- 完整副词条编码：`(1 << substat_number) | (1 << (13 + value_number))`

用途：

- 阻止一个声骸重复选择同类副词条
- 生成 `substat_all`
- 做目标词条筛选和双暴判断

## 页面中的使用点

- `Echo.vue`：新增或编辑完整声骸
- `Substat.vue`：记录单条调谐
- `SubstatAnalysis.vue`：展示统计结果
- `FindEcho.vue`：按词条组合筛选声骸
- `EchoDcritCount.vue`：双暴档位分布

## 维护原则

- 若后端变更副词条定义，前端常量必须同步更新
- `name` / `desc` 主要用于展示，统计逻辑仍以编号和位图为准
