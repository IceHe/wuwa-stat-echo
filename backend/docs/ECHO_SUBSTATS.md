# 鸣潮声骸副词条文档

## 概览

当前系统内置了 13 种副词条类型，并为每种副词条维护固定档位集合，用于：

- 单次调谐日志记录
- 档位分布统计
- 目标词条分析
- 角色模板评分

定义来源：[consts.py](../consts.py)

## 编码规则

系统使用位运算编码一个副词条及其档位：

- 低 13 位：副词条类型
- 再往上的位：档位位置

编码公式：

```text
(1 << substat_number) | (1 << (13 + value_number))
```

例如：

- `暴击 6.3%` = `1 << 0 | 1 << 13`
- `暴击伤害 12.6%` = `1 << 1 | 1 << 13`

## 相关衍生字段

### `substat1` ~ `substat5`

完整保存 5 个副词条的编码值，既能还原类型，也能还原档位。

### `substat_all`

只保留副词条类型位图，用于：

- 判断是否已存在重复词条
- 查询是否包含目标词条组合
- 统计双暴等组合命中情况

常见组合：

- 双暴：`0b11`
- 暴击 + 暴击伤害 + 攻击：`0b111`

## 评分与统计中的使用方式

### 统计

- `wuwa_tune_log.substat` 记录类型编号
- `wuwa_tune_log.value` 记录档位编号
- `/tune_stats` 汇总副词条总体概率、档位分布、孔位分布

### 评分

评分模板会按角色权重放大不同副词条价值，例如：

- 暴击权重默认更高
- 攻击百分比高于攻击固定值
- 特定角色会为共鸣效率、技能伤害、普攻等追加权重

## 前端展示约定

- `value_desc`：仅档位值，如 `8.1%`
- `value_desc_full`：完整文案，如 `暴击 8.1%`
- `s1_desc` ~ `s5_desc`：通常直接使用完整文案渲染按钮文本
